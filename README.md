# ProofArena

ProofArena 是一个高中数学解法竞技场。同一道题的标准解、结构解、考场解与讲解路线可以在统一维度下比较，并保留“怎么想到”、关键转化和验证信息。

当前仓库是可运行的静态 Demo，第一批内容聚焦 2026 天津卷数学第 16-20 题。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```
