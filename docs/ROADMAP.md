# 路线图

路线图描述方向，不代表固定发布时间。当前项目已经越过纯静态 Demo 阶段，重点是把“高质量题解样板”整理成可以投稿、审核、比赛和长期沉淀的社区产品。

## 当前阶段：社区与比赛 MVP

Proof Graph 的超级改造先按 [Proof Graph MVP 执行指挥文档](./PROOF_GRAPH_MVP_EXECUTION.md) 推进：先完成 3 道旗舰题的思路入口、解法比较、方法边界、推理回放、挑战关系和审核沉淀，不直接展开完整图数据库或多端入口。

## 暂不优先

在内容规范、审核流程和比赛闭环稳定前，暂不优先：

- 排位积分和强社交系统
- 实时聊天
- 付费功能
- 大规模自动导入题库
- LLM 自动生成大量题解

ProofArena 宁可少而可信，也不以低质量自动生成内容换取题量。
